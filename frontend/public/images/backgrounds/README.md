# Background Images

## Required Image

### background-1.png

**Location:** `frontend/public/images/backgrounds/background-1.png`

**Description:** 
- Blue sky with white clouds
- Warm, transparent, calm atmosphere
- Professional and peaceful mood

**Specifications:**
- Format: PNG
- Recommended dimensions: 1920x1080 or higher
- Quality: High resolution for web

**How to add:**
1. Place your `background-1.png` image in this directory
2. The image will be automatically loaded by the WelcomePage component
3. Path used in code: `/images/backgrounds/background-1.png`

## Alternative Solutions

If you don't have the image yet, you can:

1. **Use a placeholder service:**
   - Unsplash: https://unsplash.com/s/photos/blue-sky-clouds
   - Pexels: https://www.pexels.com/search/blue%20sky%20clouds/

2. **Temporary placeholder:**
   - Download any blue sky with clouds image
   - Rename it to `background-1.png`
   - Place it in this directory

## Testing

After adding the image, restart the development server:

```powershell
npm run dev
```

Then navigate to `http://localhost:5173` to see the welcome page with the background.
